# Phishing Email Analysis

## Project Overview
Analyzed real phishing email samples to identify malicious indicators of compromise.

## Rooms Completed
- Phishing Emails 1 – TryHackMe ✅ 100%
- Phishing Emails in Action – TryHackMe ✅ 100%

## Key Findings
- Email spoofing using mismatched sender and Reply-To fields
- Malicious URLs defanged using CyberChef
- Fake Outlook login page used for credential harvesting
- Malicious PDF attachment identified

## MITRE ATT&CK Mapping
- T1566.001 – Phishing: Spearphishing Attachment
- T1566.002 – Phishing: Spearphishing Link
- T1056.003 – Input Capture: Web Portal Capture

## Tools Used
- TryHackMe
- CyberChef
- Mozilla Thunderbird
